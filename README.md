# ros-one-click-installation
Rapid development of ROS

# Install ROS
## ros kinetic
- Replace password with your password.
  ```bash
  ./ros_kinetic_install.sh password
  ```
## ros melodic
- Replace password with your password.
  ```bash
  ./ros_melodic_install.sh password
  ```
## ros noetic
- Replace password with your password.
  ```bash
  ./ros_noetic_install.sh password
  ```
